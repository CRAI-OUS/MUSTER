class pymuster.core.utils.RunningMeanVar(n=10)

Bases: object

This class is used to calculate the mean and variance of the n last values .. rubric:: Example

>>> rmv = RunningMeanVar(n=10)
>>> for i in range(100):
>>>     rmv.add(i)
>>>     print(rmv.mean())
Parameters:
